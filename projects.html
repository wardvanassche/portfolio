CLE4

CLE4, game development, een periode van 2 weken waarin we binnen korte tijd een game moesten designen en bouwen.

De game is gemaakt met Javascript en de framework van Excalibur. We hebben een zeilgame gebouwd gebaseerd op een locatie in Rotterdam, de Kralingse Plas. Het is een explorer game met challenge aspecten, als doelgroep worden jongeren die willen leren zeilen aangesproken.

Het doel van de game is om het zeil goed te zetten waardoor je gaat varen, en vervolgens alle boeien te pakken om zo punten te verdienen.

CLE3

CLE3 stond in het teken van het bouwen van een web API met Javascript. In teamvorm hebben we een webapplicatie ontworpen en gebouwd voor mensen met een motorische beperking, met als thema "zorg". Het directe contact met de doelgroep was erg prettig, na een interview en usability-test zijn we op een mooi eindresultaat gekomen.

De applicatie is gemaakt om toegankelijke zorg voor motorisch beperkten beter vindbaar te maken. Op de beginpagina is een keuze te maken tussen verschillende soorten zorginstellingen. Vervolgens wordt een overzicht van alle nabije zorginstellingen weergegeven, met daarbij gegevens, adres, routebeschrijving en beoordelingen van andere mensen met een motorische beperking, zodat er een goed beeld ontstaat van de voorzieningen voor beperkten bij de desbetreffende locatie.

CLE2

In CLE2 zijn we individueel aan de slag gegaan. We moesten zelf op zoek naar een échte opdrachtgever om een reserveringssysteem in de vorm van een webpagina voor te maken. Ik ben terecht gekomen bij mijn kapper en heb daarvoor een website met reserveringssysteem voor gemaakt.

De website bevat een formulier waar klanten afspraken kunnen inplannen. Verder is er een login voor medewerkers, waarna je op de admin-pagina terecht komt. Op deze admin-pagina kan de opdrachtgever alle afspraken zien, de klantendatabase en zijn er extra functies zoals een optie om je wachtwoord te veranderen.

Het hele systeem is verbonden aan een database in PHPmyadmin. Deze periode hebben we veel gebruik gemaakt van PHP en SQL. Deze twee talen in combinatie met HTML/CSS zijn essentieel voor een dynamische website, aangezien in dit tijdperk vrijwel elke website enigzins dynamisch is.

CLE1 - Sprint 2&3

In sprint 2 en 3 werkten we in teamverband verder aan één van de robots uit sprint 1. Hiervoor hebben we de robot van een van onze teamgenoten gekozen. Deze robot heeft als doel om wielrenners te verjagen. Wat begon als een grap is uiteindelijk een project geworden waar veel tijd in is gestoken.

De robot bevat een fietsmand die aan de fiets kan worden bevestigd, met een deksel om hem netjes af te sluiten. In de fietsmand zit een dubbele bodem, met onderop electronica dat het spel op de fietsmand verzorgd. Verder is er een slagboom aan de fiets bevestigd die door middel van de knoppen aan het stuur bedient kan worden. Het idee van deze slagboom is dat je hem open of dicht kan doen zodra er een vervelende weggebruiker jou wilt passeren.

Deze periode hebben we vooral in het stadslab doorgebracht, waardoor ik veel ervaring heb opgedaan met de lasercutter. Daarnaast hebben we ook veel gebruik gemaakt van lijmpistolen en hebben we veel gesoldeerd.

3 okt. 2022

CLE1 - Sprint 1

Dit project eerste project stond in het teken van een robot ontwerpen voor een medestudent. Aan de hand van de Adafruit Circuit Playground Express hebben we deze robot geprogrammeerd zodat hij gaat rijden zodra hij geluid detecteerd. Hiervoor hebben we Javascript gebruikt.

Deze eerste sprint heb ik voor het eerst kennis gemaakt met Javascript en dit goed kunnen oefenen op de robot. Daarnaast heb ik een speedcourse lasercutten gevolgd, waarmee ik vervolgens de robot auto gebouwd heb uit hout.
